# Vega Dialer
AI outbound cold calling system for Vega Estimating AI — powered by Retell AI and Bland.ai.

## Agent
**Sarah** — AI outbound sales rep pitching Vega Estimating AI to construction subcontractors.

## Goal
Get subcontractors to visit [vegaestimating.ai](https://vegaestimating.ai) and book a free 10-minute live demo.

## Files
- `retell_campaign.py` — Retell AI campaign launcher
- `vt_me_bland_campaign.py` — Bland.ai campaign for VT/ME contacts
- `bland_prompt_config.json` — Sarah's prompt and call settings
- `sarah_retell_prompt.md` — Full Retell prompt for Sarah
- `cold_call_script_draft.md` — Cold call script drafts
- `email_sequence.md` — Follow-up email sequence

## Notes
- 1.75-second delay before Sarah speaks after call connects
- Only mentions the specific trade tied to each contact record
- Directs all interested leads to vegaestimating.ai
